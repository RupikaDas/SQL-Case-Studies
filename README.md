# SQL Case Studies

A collection of real-world SQL investigations focused on Trust & Safety, Fraud Detection, Platform Integrity, Risk Analytics, and User Behavior Analysis.

## Objective

This repository demonstrates practical SQL skills through business-driven case studies designed to simulate analytical investigations performed by Data Analysts, Risk Analysts, Trust & Safety Analysts, Product Analysts, and Engineering Analysts.

Rather than isolated SQL exercises, each project follows an end-to-end investigation workflow that includes problem definition, data exploration, analysis, risk scoring, prioritization, and business recommendations.
